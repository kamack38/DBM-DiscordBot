## Downloads

### Adding bot to Glitch
1. Go to [glitch.com](https://glitch.com/) and login or create account.
2. Click New Project and then "hello express".
3. Delete all files. PS. You cant delete "assets".
4. Click "Tools", "Import and Export" and then Import from GitHub.
5. Paste "kamack38/iPlay-Poland-Bot" in the pop out.
6. Go to "data" folder and then to setting.json .
7. Paste your bot token and client.
8. Click "Tools", "Terminal" and paste "enable-pnpm".
9. Your bot is now ready to use.

### Making bot work 24/7
1. In glitch.com click "Share". Then select "Live App" and copy link.
2. Go to [uptimerobot.com](https://uptimerobot.com/) and Login or Sign-up.
3. Got to dashboard.
4. Click "+ Add New Monitor".
5. Select in "Monitor Type" "HTTP(s)"
6. Write your friendly name. PS. You can write what you want.
7. In "URL (or IP)" paste link wchich you copied before.
8. Click two times "Create Monitor".
9. Your bot now work 24/7.

## Commands :

- help
  - command - show help for commands
  - moderation - show help for moderation commands
  - fun - show help for fun commands

### Moderation Commands :
- kick @user <reason> - kicks user from srver
- ban @user <reason> - bans user
- unban <user id> - unbans user
- bans - show banned users
- mute @user - mutes user
- unmute @user - unmutes user
- tempmute @user <time> - mutes user for selected time
- warn @user - warns user
- clear-warn @user - deletes all user warns
- clear <number> - deletes <number> mesages

### Info Commands :

- server-info - show server info
- user-info <@user | no user (us)> - show info about mentioned user 
- role-info @role - show info about a role

### Bot Owner Commands :
- bot-restart - restarts bot
- bot-stop - stops bot
